🚗 Test Drive Booking System

This project is a web-based vehicle test drive booking system built with Core PHP, MySQL, HTML, and CSS.

Features:
- User registration and login
- Browse available vehicles
- Book test drives with preferred date and time
- View booking history and status
- Submit feedback and reviews after completed test drive

Admin Features:
- Admin login
- Manage users
- Add, edit, delete vehicles
- Approve, reject, and complete bookings
- View customer feedback and reviews

Technologies Used:
- PHP (Core PHP with mysqli)
- MySQL
- HTML
- CSS
- XAMPP

Purpose:
This project was developed as a diploma computer engineering academic project to demonstrate CRUD operations, authentication, booking workflow, file uploads, and database management.
